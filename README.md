# Description
This week, we were tasked to create an application called `U-Develop-It`. This application in centered around MySQL and CRUD opperations. To be successful we must consider the following:

- Use MySQL to create a database.

- Use SQL to create and populate a table of data.

- Run SQL queries to create, read, update, and delete data from a table.



# Install
